## Descripción

Un juego casual donde el jugador debe **atrapar estrellas** que caen del cielo.  
- Cada estrella atrapada = **+1 punto**  
- Si no la atrapas, desaparece  
- **30 segundos por partida**  
- Efectos de chispa al hacer clic

---

## Controles

| Acción         | Teclado / Móvil |
|----------------|-----------------|
| Atrapar estrella | **Clic / Toque** |
| Reiniciar       | Botón "Jugar de Nuevo" |

## Tecnologías

## Cómo jugar

1. Abre el enlace
2. Haz clic en **"Jugar"**
3. **Toca o haz clic** en las estrellas que caen
4. ¡Supera tu récord!
